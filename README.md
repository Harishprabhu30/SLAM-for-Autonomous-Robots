# SLAM-for-Autonomous-Robots

# Autonomous Systems:

Autonoumous Systems are those systems which are developed to function without the interference of humans and moreover, it has an added benefit of working with the humans (like HRC - Human Robot Collaboration). These systems have semantic understanding, which enables them to understand static objects, dynamic objects in the environment.

Examples of the autonomous systems are Robots deployed in various industries for instance, Rovers for space exploration, Kiva - amazon developed robot for warehouse, robots developed for deliveries, and the most noticable and famous self driving cars developed by Tesla, Google.

To achieve a system to work autonomously, there are several algorithms, models working in its embedded system. Here, I am glad to work on one of the algorithm that is crucial and used widely in these systems.

# SLAM - Simultaneous Localization and Mapping

SLAM is a concept where an autonomous system is provided with the ability to navigate and at the same time, it can locate itself in the environment where it is placed with little or without the intervention of human.
